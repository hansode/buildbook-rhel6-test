Test Buildbook for RHEL6
========================

Test [buidbook-rhel6](https://github.com/hansode/buildbook-rhel6) building

Requirements
------------

+ RHEL/CentOS/Scientific

Getting Started
---------------

Setup submodules.

```
$ make setup
```

Test Build for specific book

```
$ ./test-print.sh <book-name>
```
